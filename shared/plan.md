## 各記事の概要

### 1. 有田焼 / Arita Porcelain
佐賀県有田町を発祥とする日本初の磁器で、400年以上の歴史を誇る。
白磁に繊細な絵付けを施した「伊万里焼」としても知られ、江戸時代にはヨーロッパ王侯貴族を魅了した。

### 2. 西陣織 / Nishijin Weaving
京都・西陣地区で生まれた高級絹織物で、複雑な紋様を先染めの糸で織り上げる技法が特徴。
平安時代から続く宮廷文化と結びつき、現在も能装束や帯地として伝統が息づいている。

### 3. 南部鉄器 / Nambu Ironware
岩手県盛岡・奥州を中心に発展した鉄器工芸で、茶釜や鉄瓶が代表的な作品として知られる。
丈夫で熱伝導率が高く、近年は機能美を持つ日用品として国際的な注目を集めている。

### 4. 江戸切子 / Edo Kiriko
東京（江戸）で生まれたカットガラス工芸で、鋭利な切子紋様が光を乱反射して輝く。
明治時代に西洋のガラス技術と日本の職人技が融合し、独自の美的様式を確立した。

### 5. 友禅染め / Yuzen Dyeing
京都発祥の手描き染色技法で、防染糊と多色の染料を組み合わせて緻密な図柄を表現する。
江戸中期に宮崎友禅斎が確立したとされ、京友禅・加賀友禅の二大流派が今も継承されている。

### 6. 木版画 / Mokuhanga Woodblock Print
江戸時代に花開いた浮世絵版画を代表とする日本の多色木版印刷技法。
ゴッホやモネら西洋印象派の画家にも影響を与え、現代の国際的なアート界でも高く評価される。

### 7. 組紐 / Kumihimo Braiding
複数の糸を規則的に組み合わせて作る日本の伝統的な紐工芸で、奈良時代以前から存在する。
刀の柄巻きや帯締めとして武家・公家の文化を支え、現在はアクセサリーとして世界的に人気がある。

### 8. 和紙 / Washi Paper
楮・三椏・雁皮などの植物繊維を手漉きで作る日本固有の紙で、ユネスコ無形文化遺産に登録されている。
強度と柔軟性を兼ね備え、書道・版画・工芸から文化財修復まで幅広い用途で現代も活躍している。

### 9. 備前焼 / Bizen Pottery
岡山県備前市を産地とする日本六古窯の一つで、釉薬を使わず高温で焼き締める技法が特徴。
土の自然な表情と炎の偶然の模様（緋襷・牡丹餅など）が生む一点物の美しさが魅力。

### 10. 金継ぎ / Kintsugi
割れや欠けた陶磁器を漆で継ぎ、金・銀・白金の粉で装飾する日本固有の修復技法。
傷を隠すのではなく美として昇華する哲学は「物の不完全さを受け入れる」侘び寂びの精神を体現している。
